📖 Technical Information

This RSA Security System was developed for Borg El Arab Technological University as a final project for IT Security.

  • Encryption Logic: Implements simulated RSA asymmetric key pairs.
  • Presentation: Interactive UI built with Glassmorphism principles.
  • Hosting: Deployed on Vercel for real-time cloud accessibility.

Designed to show how sensitive data can be protected during network transmission.
