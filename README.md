# Secure Messaging System (RSA Encryption)

A professional web-based security project developed for **Borg El Arab Technological University**. This application demonstrates how sensitive data can be protected during transmission using asymmetric encryption.

## 🛡️ Project Overview
This project implements a secure communication channel where messages are encrypted using the **RSA (Rivest–Shamir–Adleman)** algorithm. It features a modern web interface to bridge the gap between complex cryptography and user-friendly interaction.

## 🚀 Features
* **Asymmetric Encryption:** Uses Public and Private key pairs for maximum security.
* **Real-time Processing:** Encryption and decryption happen instantly via a Node.js backend.
* **Responsive UI:** A clean, Arabic-supported interface for ease of use.
* **RESTful API:** Communicates between Frontend and Backend using JSON.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+).
* **Backend:** Node.js, Express.js.
* **Cryptography:** Node.js built-in `crypto` module.
